Photon: Particle Nature of Light Core Foundation

Einstein's revolutionary idea (1905, Nobel Prize 1921): light is not a continuous wave but a stream of discrete packets of energy called photons. Each photon carries energy proportional to frequency, not amplitude. This directly explains why blue light ejects electrons from caesium but red light cannot — regardless of intensity.

Photon Definition

A photon is a massless quantum of electromagnetic radiation. It travels at speed $c$ in vacuum, carries energy $E = h\nu$ and momentum $p = h/\lambda = h\nu/c$. It has zero rest mass but non-zero relativistic momentum.

Photon Energy
  • $E = h\nu = hc/\lambda$
  • $h = 6.626 \times 10^{-34}$ J·s
  • $hc = 1240$ eV·nm (memorise!)
  • $E$ (in eV) $= 1240/\lambda$ (nm)
Photon Momentum
  • $p = h/\lambda = E/c = h\nu/c$
  • Perfectly reflecting surface: $\Delta p = 2p = 2h/\lambda$
  • Perfectly absorbing: $\Delta p = h/\lambda$
  • Photon has NO charge, NO rest mass
Number of Photons
  • $n = P/E = P\lambda/(hc)$
  • $P$ = power of source (W)
  • Intensity $I \propto n$ (not frequency)
  • Photon count per second = $n$

$hc = 1240$ eV·nm — The single most important constant combination in Modern Physics. Memorise it cold. With this, you can calculate photon energy from wavelength in 2 seconds: $E(\text{eV}) = 1240/\lambda(\text{nm})$. Example: $\lambda = 400$ nm → $E = 3.1$ eV. No calculator needed.

🎯 PATTERN 1: Photon Energy Calculation

What NEET tests: Given wavelength (or frequency), find energy in eV. Or given power and wavelength, find number of photons per second.

PYQ 2024: Light of wavelength $\lambda = \frac{hc}{e}$ metre is incident on a photosensitive surface. If the work function is $\phi$ eV, the stopping potential is:

Step 1: Find photon energy. $E = hc/\lambda$. Given $\lambda = hc/e$ metres, so $E = hc/(hc/e) = e$ joules $= 1$ eV.

Step 2: Apply Einstein's equation: $eV_s = E - \phi \Rightarrow V_s = 1 - \phi$ volts.

If $\phi < 1$ eV, stopping potential = $(1-\phi)$ V. If $\phi \geq 1$ eV, no emission.

Photoelectric Effect — Einstein's Equation Most Tested

When light of frequency $\nu > \nu_0$ (threshold) falls on a metal surface, electrons are ejected. The key insight: a single photon interacts with a single electron — this is a one-to-one collision. Extra photon energy beyond the work function becomes the electron's kinetic energy.

Einstein's Photoelectric Equation
$KE_{max} = h\nu - \phi_0 = h\nu - h\nu_0$
$eV_s = h\nu - \phi_0$
$V_s = \frac{h\nu - \phi_0}{e} = \frac{h(\nu - \nu_0)}{e}$
Work Function $\phi_0$

Minimum energy needed to eject an electron from the metal surface. $\phi_0 = h\nu_0 = hc/\lambda_0$. Measured in eV. Depends only on the metal, not on incident light.

Threshold Frequency $\nu_0$

Minimum frequency of light that can eject electrons. $\nu_0 = \phi_0/h$. Below $\nu_0$, no emission regardless of intensity. $\lambda_0 = hc/\phi_0$.

Stopping Potential $V_s$

The retarding potential needed to stop the most energetic photoelectron. $eV_s = KE_{max}$. Does NOT depend on intensity — only on frequency and work function.

Saturation Current

Maximum photoelectric current when all emitted electrons are collected. Proportional to intensity (more photons → more electrons). Independent of frequency (above $\nu_0$).

🚨 #1 NEET Trap — Intensity vs Frequency

Students confuse intensity with frequency. Intensity affects only the NUMBER of electrons ejected, NOT their kinetic energy. $V_s$ (stopping potential) is independent of intensity. Doubling intensity doubles current, but stopping potential stays the same. This distinction is tested almost every year.

🎯 PATTERN 2: Stopping Potential Calculation (Highest Frequency)

What NEET tests: Electron in hydrogen atom jumps from state $n$ to $n=1$, emitting a photon. This photon irradiates a metal with work function $\phi$. Find stopping potential.

Step 1: Find photon energy using Bohr's formula: $E = 13.6\left(\frac{1}{n_1^2} - \frac{1}{n_2^2}\right)$ eV (for H-atom).

Step 2: Apply Einstein: $eV_s = E_{photon} - \phi_0$.

Step 3: $V_s = (E_{photon} - \phi_0)/e$ volts. Note: if $E_{photon} < \phi_0$, no emission.

PYQ (Classic — appeared 3 times): The electron in H-atom jumps from $n=3$ to $n=1$. The emitted photon irradiates a photosensitive material with work function 5.1 eV. Find stopping potential. [$E_n = -13.6/n^2$ eV]

Step 1: $E_{photon} = E_3 - E_1 = \left(-\frac{13.6}{9}\right) - (-13.6) = -1.511 + 13.6 = 12.09$ eV $\approx 12.1$ eV.

Step 2: $eV_s = 12.1 - 5.1 = 7.0$ eV → $V_s = 7$ V.

Key check: $12.1 > 5.1$, so emission occurs. Answer: 7 V.

PYQ 2010: Monochromatic radiation emitted when electron in H-atom jumps from first excited state (n=2) to ground state (n=1) irradiates a photosensitive material. Stopping potential = 3.57 V. Find threshold frequency.

Step 1: $E_{photon}$ for $n=2 \to n=1$: $\Delta E = 13.6(1/1^2 - 1/2^2) = 13.6 \times 3/4 = 10.2$ eV.

Step 2: $\phi_0 = E_{photon} - eV_s = 10.2 - 3.57 = 6.63$ eV.

Step 3: $\nu_0 = \phi_0/h = (6.63 \times 1.6 \times 10^{-19})/(6.626 \times 10^{-34}) \approx 1.6 \times 10^{15}$ Hz.

🎯 PATTERN 3: Frequency Doubled — New KEmax

PYQ (repeated pattern): Work function of metal is $\phi$. Incident frequency $\nu$ gives KEmax $= E$. If frequency is doubled, find new KEmax.

Original: $E = h\nu - \phi \Rightarrow h\nu = E + \phi$.

Doubled frequency: $KE' = h(2\nu) - \phi = 2h\nu - \phi = 2(E+\phi) - \phi = 2E + \phi$.

Answer: $KE' = 2E + \phi$. This is NOT $2E$ — the trap is forgetting that $\phi$ is subtracted only once.

Frequency doubled trick: If $\nu \to 2\nu$, then $KE_{new} = 2KE_{old} + \phi_0$. Never $2KE_{old}$.

Wavelength halved trick: Same as doubling frequency, so same formula applies.

Graphical Analysis — Vs vs Frequency Graph

The graph of stopping potential ($V_s$) vs frequency ($\nu$) is a straight line: $V_s = (h/e)\nu - \phi_0/e$. The slope is $h/e$ — the same for ALL metals. The x-intercept (where $V_s = 0$) gives $\nu_0$ — different for each metal.

✅ Key Insight from Graph

All metals give parallel lines (same slope = $h/e$) but with different x-intercepts ($\nu_0$ values). The slope of this graph was historically used to measure Planck's constant $h$ accurately. NEET asks: "From the graph, which quantity is the slope?" — Answer: $h/e$.

de Broglie Hypothesis — Matter Waves 3rd Most Tested

Louis de Broglie's 1924 hypothesis: every moving particle has an associated wavelength. This was the reverse of Einstein's photon idea — if waves can behave as particles, particles can behave as waves. Verified experimentally by Davisson and Germer (1927) through electron diffraction from a nickel crystal.

de Broglie Wavelength
$\lambda = \frac{h}{p} = \frac{h}{mv}$

For particle accelerated through voltage $V$:
$\lambda = \frac{h}{\sqrt{2mqV}}$

For particle at temperature $T$:
$\lambda = \frac{h}{\sqrt{2m k_B T}} = \frac{h}{\sqrt{3m k_B T}}$ (rms speed)

🎯 PATTERN 4: Comparing de Broglie Wavelengths (Most Common!)

NEET repeatedly asks: compare $\lambda$ for electron, proton, neutron, alpha particle — all accelerated through the same voltage $V$, or all having the same kinetic energy.

For same voltage $V$: $\lambda = h/\sqrt{2mqV}$. Here $q$ is the charge. Write $\lambda \propto 1/\sqrt{mq}$.

For same kinetic energy $KE$: $p = \sqrt{2m \cdot KE}$, so $\lambda = h/\sqrt{2m \cdot KE} \propto 1/\sqrt{m}$. Lighter particle → longer wavelength.

For same momentum $p$: $\lambda = h/p$ → all have the same wavelength (obvious!)

PYQ 2024 (Re-exam): Electron and alpha particle are accelerated by same potential difference. Let $\lambda_e$ and $\lambda_\alpha$ denote their de Broglie wavelengths. Then:

(a) $\lambda_e > \lambda_\alpha$ (b) $\lambda_e = 4\lambda_\alpha$ (c) $\lambda_e = \lambda_\alpha$ (d) $\lambda_e < \lambda_\alpha$

$\lambda \propto 1/\sqrt{mq}$. For electron: $m_e q_e$ is very small. For alpha: $m_\alpha = 4m_p \approx 7280 m_e$, $q_\alpha = 2e$. So $m_\alpha q_\alpha \gg m_e q_e$.

Therefore $\lambda_e \gg \lambda_\alpha$. Answer: (a) $\lambda_e > \lambda_\alpha$.

PYQ 2025: A photon and an electron (mass $m$) have the same energy $E$. The ratio $\lambda_{photon}/\lambda_{electron}$ is: ($c$ = speed of light)

(a) $\sqrt{2mE}/c \cdot h^{-1}$    (b) $c\sqrt{2m/E}$    (c) $\sqrt{E/(2mc^2)}$    (d) $\sqrt{2mE} \cdot c$

Photon: $E = hc/\lambda_{photon} \Rightarrow \lambda_{photon} = hc/E$.

Electron: $E = p^2/(2m) \Rightarrow p = \sqrt{2mE} \Rightarrow \lambda_{electron} = h/\sqrt{2mE}$.

Ratio: $\frac{\lambda_{photon}}{\lambda_{electron}} = \frac{hc/E}{h/\sqrt{2mE}} = \frac{c\sqrt{2mE}}{E} = c\sqrt{2m/E}$.

Answer: (b) $c\sqrt{2m/E}$.

de Broglie Wavelength for Particle at Temperature T

For particles in thermal equilibrium at temperature $T$: using kinetic theory, $KE = \frac{3}{2}k_BT$ (for rms) or $KE = \frac{3}{2}k_BT$ depending on context. NEET uses $\lambda = h/\sqrt{2mk_BT}$ for most problems (using average KE = $k_BT$) or sometimes the full $3k_BT/2$ version.

Neutron in thermal equilibrium (NEET 2013): $\lambda = h/\sqrt{2mk_BT}$. For neutron: $m = 1.67\times10^{-27}$ kg, plugging in gives $\lambda = 30.8/\sqrt{T}$ Å. This exact numerical result has appeared in NEET — memorise it.

PYQ 2013: de Broglie wavelength of neutrons in thermal equilibrium at temperature $T$ is:

(a) $3.08/\sqrt{T}$ Å    (b) $0.308/\sqrt{T}$ Å    (c) $0.0308/\sqrt{T}$ Å    (d) $30.8/\sqrt{T}$ Å

$\lambda = h/\sqrt{2mk_BT}$. Plugging: $h = 6.63\times10^{-34}$, $m = 1.67\times10^{-27}$ kg, $k_B = 1.38\times10^{-23}$ J/K.

$\lambda = \frac{6.63\times10^{-34}}{\sqrt{2 \times 1.67\times10^{-27} \times 1.38\times10^{-23} \times T}} = \frac{6.63\times10^{-34}}{\sqrt{4.608\times10^{-50}T}} = \frac{6.63\times10^{-34}}{6.789\times10^{-25}\sqrt{T}}$

$= 9.77\times10^{-10}/\sqrt{T}$ m $\approx 30.8/\sqrt{T}$ Å. Answer: (d).

Tip: Just memorise the final result — $30.8/\sqrt{T}$ Å for neutron.

Electron Emission Mechanisms Low Frequency

Thermionic Emission

Metal heated to high temperature. Thermal energy overcomes work function. Used in cathode ray tubes, electron guns. Example: tungsten filaments.

Photoelectric Emission

Light of $\nu > \nu_0$ falls on metal surface. Photon energy overcomes work function. Basis of photodetectors, solar cells.

Field Emission

Strong electric field (~$10^8$ V/m) applied. Quantum tunnelling allows electrons to escape without thermal energy. Used in field emission displays.

Secondary Emission

High-energy electrons bombard metal surface, ejecting secondary electrons. Used in photomultiplier tubes. Ratio of secondary to primary electrons is secondary emission ratio.

📝 NEET 2025 Specific

In the Davisson-Germer experiment, electrons showed diffraction peaks at specific angles — confirming wave nature of electrons. The experiment used a nickel crystal as diffraction grating. The accelerating voltage was 54 V → $\lambda = 1.67$ Å, matching Bragg diffraction peak at 50°. Key fact: this was the first direct experimental proof of de Broglie hypothesis.

NEET Trap Analysis — Dual Nature Exam Survival

1
❌ Wrong Think

"Increasing intensity of light above threshold increases stopping potential"

✓ Correct

Intensity increases only the number of photoelectrons (current). $V_s$ depends only on frequency: $eV_s = h\nu - \phi_0$

2
❌ Wrong Think

"de Broglie wavelength depends on charge and nature of particle"

✓ Correct

$\lambda = h/mv$ — depends on mass and velocity (momentum). Charge does NOT appear in de Broglie formula. (NEET 2024 Statement I was incorrect)

3
❌ Wrong Think

"Photon has positive charge" or "photon mass = $h\nu/c^2$"

✓ Correct

Photon has zero charge and zero REST mass. Its energy-momentum relation is $E = pc$ (massless particle). $m = 0$, not $h\nu/c^2$.

4
❌ Wrong Think

"For same KE, heavier particle has shorter de Broglie wavelength"

✓ Correct

$\lambda \propto 1/\sqrt{m}$ for same KE. YES — heavier particle → shorter $\lambda$. But for same MOMENTUM, all have same $\lambda$. Distinguish the condition!

Chapter 1 — Last 5-Minute Recall

  • $E_{photon} = h\nu = hc/\lambda$. Use $hc = 1240$ eV·nm always.
  • Einstein: $KE_{max} = h\nu - \phi_0 = eV_s$
  • $V_s \propto \nu$, $I_{sat} \propto$ Intensity. These are independent.
  • $\lambda_{dB} = h/p = h/\sqrt{2mKE} = h/\sqrt{2mqV}$
  • Same voltage: $\lambda \propto 1/\sqrt{mq}$ → electron has longest $\lambda$
  • Same KE: $\lambda \propto 1/\sqrt{m}$ → lightest has longest $\lambda$
  • Photon: $p = E/c = h/\lambda$, zero charge, zero rest mass

Rutherford's Nuclear Model Foundation Context

Geiger-Marsden alpha scattering experiment (1909–1911): alpha particles fired at thin gold foil. Most passed through, very few bounced back at large angles. This proved:

  • Atom is mostly empty space
  • All positive charge and most mass is concentrated in a tiny nucleus (~$10^{-15}$ m)
  • Electrons orbit around the nucleus
Distance of Closest Approach (r₀)

When an alpha particle ($Z_1 = 2$) approaches a nucleus ($Z_2$) head-on and momentarily stops: all KE converts to electrostatic PE. $r_0 = \frac{2kZ_1Z_2 e^2}{mv^2} = \frac{kZ_1Z_2e^2}{KE}$. This gives the nuclear diameter estimate.

PYQ (Pattern): When an alpha particle (energy 5 MeV) bombards a gold nucleus ($Z = 79$), find the distance of closest approach.

At closest approach: $KE = PE \Rightarrow 5 \text{ MeV} = \frac{kq_1q_2}{r_0}$

$r_0 = \frac{9\times10^9 \times 2 \times 1.6\times10^{-19} \times 79 \times 1.6\times10^{-19}}{5 \times 10^6 \times 1.6\times10^{-19}}$

$r_0 = \frac{9\times10^9 \times 2 \times 79 \times (1.6)^2 \times 10^{-38}}{8\times10^{-13}} \approx 4.55 \times 10^{-14}$ m $\approx 45.5$ fm

⚠ Rutherford's Model Failure

An orbiting (accelerating) electron should continuously radiate energy, spiralling into the nucleus in ~$10^{-8}$ s. The model predicts no stable atoms and no discrete spectra — both contradicted by experiment. This is why Bohr's model was necessary.

Bohr's Model — The Three Postulates Most Asked

Bohr (1913) fixed Rutherford's model with three revolutionary postulates that worked perfectly for hydrogen:

  • Stable Orbits: Electrons can orbit in certain allowed circular orbits without radiating energy. In these orbits they have fixed energy (stationary states).
  • Angular Momentum Quantisation: Orbital angular momentum $L = mvr = n\hbar = nh/2\pi$ where $n = 1, 2, 3, \ldots$ (principal quantum number).
  • Quantum Radiation: When an electron jumps from orbit $n_2$ to $n_1$ ($n_2 > n_1$), it emits a photon of energy $E = h\nu = E_{n_2} - E_{n_1}$.

All Bohr Formulas — Complete and Derivation-Aware

For Hydrogen Atom ($Z=1$) and Hydrogen-like Ion (atomic number $Z$)

Radius of $n$th orbit: $r_n = \frac{n^2 a_0}{Z}$ where $a_0 = 0.529$ Å (Bohr radius)
$r_n = \frac{0.529 \times n^2}{Z}$ Å = $\frac{n^2}{Z} \times 0.529$ Å

Speed in $n$th orbit: $v_n = \frac{v_0 Z}{n}$ where $v_0 = 2.18 \times 10^6$ m/s

Energy of $n$th orbit: $E_n = -\frac{13.6 Z^2}{n^2}$ eV

Time period: $T_n \propto n^3/Z^2$ (since $T = 2\pi r/v$)
Frequency of revolution: $f_n \propto Z^2/n^3$
Angular momentum: $L_n = n\hbar = nh/2\pi$
Scaling Laws ($\propto$)
  • $r_n \propto n^2/Z$
  • $v_n \propto Z/n$
  • $E_n \propto -Z^2/n^2$
  • $T_n \propto n^3/Z^2$
  • $L_n \propto n$
  • $KE_n = -E_n = 13.6Z^2/n^2$ eV
  • $PE_n = 2E_n = -27.2Z^2/n^2$ eV
H-atom Energy Levels
  • $E_1 = -13.6$ eV (ground)
  • $E_2 = -3.4$ eV
  • $E_3 = -1.51$ eV
  • $E_4 = -0.85$ eV
  • $E_5 = -0.544$ eV
  • $E_\infty = 0$ (ionised)
  • IE from ground = 13.6 eV
Key Transitions (eV)
  • $n=2\to1$: 10.2 eV (Lyman)
  • $n=3\to1$: 12.09 eV
  • $n=4\to1$: 12.75 eV
  • $n=3\to2$: 1.89 eV (Balmer)
  • $n=4\to2$: 2.55 eV
  • $n=4\to3$: 0.66 eV (Paschen)
  • $n=\infty\to1$: 13.6 eV (series limit)

KE and PE in Bohr orbits: $KE = -E_n$ (positive), $PE = 2E_n$ (negative). Total $E = KE + PE = -KE$. So for $E_1 = -13.6$ eV: $KE_1 = +13.6$ eV, $PE_1 = -27.2$ eV. This exact relation (PE = 2E) comes from the virial theorem for Coulomb force.

🎯 PATTERN 5: Number of Spectral Lines from Excited State

If an atom is excited to state $n$, the number of possible spectral lines when it de-excites to ground state is $\binom{n}{2} = \frac{n(n-1)}{2}$. This counts all possible transitions, not just the direct ones.

Number of spectral lines from $n$th excited state

$N = \frac{n(n-1)}{2}$

From 4th excited state ($n=5$): $N = \frac{5 \times 4}{2} = 10$ lines

From 3rd excited state ($n=4$): $N = \frac{4 \times 3}{2} = 6$ lines

⚠ Careful: n vs "nth excited state"

Ground state is $n=1$. 1st excited state is $n=2$, 2nd is $n=3$, 3rd is $n=4$. So "$n$th excited state" = principal quantum number $n+1$. Number of lines from $n$th excited state = $\frac{(n+1)n}{2}$. Don't confuse "excited state number" with principal quantum number $n$.

PYQ (multiple times): H-atom in ground state is excited by radiation $\lambda = 975$ Å. Number of spectral lines emitted?

$E_{photon} = hc/\lambda = 1240\text{ eV·nm}/(97.5\text{ nm}) = 12.72$ eV.

H-atom in ground state: $E_1 = -13.6$ eV. After absorbing: $E_{final} = -13.6 + 12.72 = -0.88 \approx -0.85$ eV $\approx E_4$. So excited to $n=4$.

Number of lines: $\frac{4 \times 3}{2} = 6$. Answer: 6 spectral lines.

🎯 PATTERN 6: Ratio of Wavelengths of Different Lines

PYQ 2017: Ratio of wavelengths of last line (series limit) of Balmer series to last line of Lyman series is:

Last (limiting) line means $n_2 = \infty$.

Balmer series limit: $1/\lambda_B = R_H(1/4 - 0) = R_H/4 \Rightarrow \lambda_B = 4/R_H$.

Lyman series limit: $1/\lambda_L = R_H(1/1 - 0) = R_H \Rightarrow \lambda_L = 1/R_H$.

Ratio: $\lambda_B/\lambda_L = 4$. Answer: 4.

PYQ 2012 (classic): Electron in H-atom jumps from 3rd orbit to 2nd orbit, emitting photon of wavelength $\lambda$. When it jumps from 4th orbit to 3rd orbit, what is the wavelength?

$3 \to 2$: $1/\lambda = R_H(1/4 - 1/9) = R_H \times 5/36 \Rightarrow \lambda = 36/(5R_H)$.

$4 \to 3$: $1/\lambda' = R_H(1/9 - 1/16) = R_H \times 7/144 \Rightarrow \lambda' = 144/(7R_H)$.

$\lambda'/\lambda = \frac{144/(7R_H)}{36/(5R_H)} = \frac{144 \times 5}{36 \times 7} = \frac{720}{252} = \frac{20}{7}$

So $\lambda' = \frac{20\lambda}{7}$.

🎯 PATTERN 7: Bohr Model — Dependence on n and Z

PYQ (NEET 2013 Karnataka): Time period in initial state is 8 times in final state. Possible values of $n_1$ and $n_2$?

$T \propto n^3$. So $T_1/T_2 = n_1^3/n_2^3 = 8 \Rightarrow n_1/n_2 = 2$.

Possible: $n_1 = 2, n_2 = 1$; $n_1 = 4, n_2 = 2$; $n_1 = 6, n_2 = 3$.

NEET options usually give $(n_1=2, n_2=1)$ — the simplest valid pair.

Hydrogen-like Ions (He⁺, Li²⁺, etc.)

For an ion with nuclear charge $Z$, simply replace $Z$ in all Bohr formulas. The energy levels scale as $Z^2$: $E_n(Z) = -13.6 Z^2/n^2$ eV. The radius scales as $1/Z$.

PYQ 2010: Ground state energy of H-atom is $-13.6$ eV. Energy of He$^+$ ion in first excited state ($n=2$) is:

He$^+$ has $Z=2$. First excited state is $n=2$.

$E_2(\text{He}^+) = -13.6 \times 4/4 = -13.6$ eV.

Wait — $E_n = -13.6Z^2/n^2 = -13.6 \times 4/4 = -13.6$ eV. Answer: $-13.6$ eV.

Note: He$^+$ in $n=2$ has same energy as H in $n=1$. This is because $Z/n$ ratio is the same.

He$^+$ shortcut: He$^+$ in $n$th state = H in $(n/Z)$th state energy-wise. Since $Z=2$ for He$^+$: He$^+$ in $n=1$ has same energy as H in $n=1/2$ (impossible), but He$^+$ in $n=2$ has energy $-13.6 \times 4/4 = -13.6$ eV = same as H ground state. He$^+$ in $n=4$ = H in $n=2$ energy-wise.

1
❌ Wrong

"Paschen series is in visible region"

✓ Correct

Paschen series ($n_1=3$) is in the infrared (IR) region. Only Balmer is in the visible region (and its first few lines). Lyman is UV.

2
❌ Wrong

"Speed of electron decreases with increasing $n$"

✓ Correct

Yes, $v \propto 1/n$ — speed DECREASES. But kinetic energy also decreases: $KE \propto 1/n^2$. Angular momentum $L = mvr \propto n$ — INCREASES. Don't confuse v, KE, and L.

3
❌ Wrong

"Maximum number of lines when atom de-excites from $n=4$ is 4"

✓ Correct

$N = n(n-1)/2 = 4\times3/2 = 6$ lines. The trap is thinking only 1 line per jump or only the direct path to ground state.

4
❌ Wrong

"Potential energy of electron in Bohr's atom is positive"

✓ Correct

PE is negative (attractive Coulomb). $PE = -2 \times |E_n|$. Total energy = KE + PE = negative. $KE = +|E_n|$, $PE = -2|E_n|$, $E_{total} = -|E_n|$.

Nuclear Properties — Size, Density, Composition Foundation

Nuclear Radius
$R = R_0 A^{1/3}$ where $R_0 = 1.2$ fm (femtometre = $10^{-15}$ m)
Nuclear volume: $V \propto A$ → nuclear density is constant (same for all nuclei!)
Composition

Nucleus = protons (Z) + neutrons (N). Mass number $A = Z + N$. Notation: ${}^A_Z X$.

Nuclear Density

$\rho_{nucleus} \approx 2.3 \times 10^{17}$ kg/m³. This is constant for ALL nuclei — the most important fact about nuclear density. $\sim 10^{14}$ times water density.

Isotopes/Isobars/Isotones

Isotopes: same Z, different A (same element). Isobars: same A, different Z. Isotones: same N, different Z. NEET tests definitions + examples every 2 years.

Nuclear Force

Strongest force in nature. Short range (~2–3 fm). Charge-independent (same for p-p, n-n, p-n). Saturates (nucleon only interacts with neighbours). Non-central.

PYQ 2012: If nuclear radius of $^{27}$Al is 3.6 fm, find approximate nuclear radius of $^{64}$Cu.

$R \propto A^{1/3}$. $R_{Cu}/R_{Al} = (64/27)^{1/3} = (64/27)^{1/3} = 4/3$.

$R_{Cu} = 3.6 \times 4/3 = 4.8$ fm. Answer: 4.8 fm.

Mass Defect & Binding Energy High Yield

The key insight: The mass of a nucleus is LESS than the sum of masses of its constituent protons and neutrons. This "missing mass" (mass defect $\Delta m$) has been converted into the energy that binds the nucleons together.

Mass Defect and Binding Energy
$\Delta m = [Zm_p + Nm_n] - M_{nucleus}$
Binding Energy: $BE = \Delta m \times c^2$
In practical units: $BE = \Delta m \times 931.5$ MeV (if $\Delta m$ in atomic mass units, u)
Binding Energy Per Nucleon: $BE/A$
✅ Key Conversion

$1$ u $= 931.5$ MeV/c². This single conversion factor converts mass defect in u to binding energy in MeV. Memorise: $\Delta m$(u) $\times 931$ = $BE$ (MeV).

BE/A Curve — The Most Important Graph in Nuclear Physics

Binding Energy Per Nucleon vs Mass Number
Reading the BE/A Curve

Fusion (light nuclei → heavier): BE/A increases → energy released. Sun's energy = fusion of H to He.
Fission (heavy nuclei → lighter): BE/A increases → energy released. $^{235}$U splits → Fe-range products with higher BE/A.
Iron-56 ($^{56}$Fe) has the highest BE/A ≈ 8.8 MeV/nucleon — most stable nucleus. Cannot release energy by either fission or fusion.

PYQ (classic): Binding energy per nucleon of D$_2$ (deuterium) and He nuclei are 1.1 MeV and 7.0 MeV respectively. If two D nuclei fuse to form He, energy released per fusion is:

$^2_1H + ^2_1H \to ^4_2He$.

BE of 2 deuterons: $2 \times (2 \times 1.1) = 4.4$ MeV.

BE of He-4: $4 \times 7.0 = 28.0$ MeV.

Energy released = $BE_{products} - BE_{reactants} = 28.0 - 4.4 = 23.6$ MeV.

Radioactivity — Decay Law, Half-life & Activity Highest Frequency

Radioactive decay is a statistical, spontaneous process. It cannot be accelerated, slowed, or stopped by any physical or chemical means (temperature, pressure, chemical state — irrelevant). The only factor is the isotope's intrinsic decay constant $\lambda$.

Radioactive Decay Law
$N(t) = N_0 e^{-\lambda t}$
Half-life: $T_{1/2} = \ln 2/\lambda = 0.693/\lambda$
$N = N_0 \left(\frac{1}{2}\right)^{t/T_{1/2}}$
Activity: $A = \lambda N = A_0 e^{-\lambda t}$
Mean life: $\tau = 1/\lambda = T_{1/2}/0.693 \approx 1.443 T_{1/2}$

$(1/2)^n$ method: After $n$ half-lives, remaining fraction $= (1/2)^n$. After 1 half-life: 50% remains. After 2: 25%. After 3: 12.5%. After 4: 6.25%. After 10: $(1/1024) \approx 0.1\%$. This method avoids exponential calculations — use it in NEET.

🎯 PATTERN 8: Half-life Calculation (Most Common Type)

Type A: Given $T_{1/2}$, find fraction remaining/decayed after time $t$.

Number of half-lives: $n = t/T_{1/2}$. Remaining: $(1/2)^n$. Decayed: $1 - (1/2)^n$.

Type B: Given initial and final amounts, find time elapsed.

$N/N_0 = (1/2)^n \Rightarrow n = \log_2(N_0/N) \Rightarrow t = n \times T_{1/2}$.

Type C: Given two decay constants, find when ratio of activities equals specific value.

Use $A_1/A_2 = (\lambda_1 N_{01} e^{-\lambda_1 t}) / (\lambda_2 N_{02} e^{-\lambda_2 t})$.

PYQ 2021: Half-life of radioactive nuclide is 100 hours. The fraction of original activity that remains after 150 hours is:

$n = t/T_{1/2} = 150/100 = 1.5$ half-lives.

$A/A_0 = (1/2)^{1.5} = (1/2)^1 \times (1/2)^{0.5} = 0.5 \times (1/\sqrt{2}) = 1/(2\sqrt{2}) = \sqrt{2}/4$.

Answer: $1/(2\sqrt{2})$ or equivalently $\sqrt{2}/4 \approx 0.354$.

PYQ (classic — multiple times): A radioactive sample has $T_{1/2} = 20$ minutes. How long to go from 40% decay to 85% decay?

40% decayed → 60% remaining: $(1/2)^{n_1} = 0.6$. → Not a nice number. Use: $N = N_0 e^{-\lambda t_1}$, $0.6 = e^{-\lambda t_1}$.

85% decayed → 15% remaining: $0.15 = e^{-\lambda t_2}$.

$\Delta t = t_2 - t_1 = \frac{\ln(0.6/0.15)}{\lambda} = \frac{\ln 4}{\lambda} = \frac{2\ln 2}{\lambda} = 2T_{1/2} = 40$ min.

Answer: 40 minutes (= 2 half-lives).

Key: $0.6/0.15 = 4 = 2^2$ → exactly 2 half-lives difference.

PYQ 2017: Radioactive material A has decay constant $8\lambda$, material B has decay constant $\lambda$. Initially same number of nuclei. After what time is ratio of B to A equal to $e$?

$N_B/N_A = e^{-\lambda t} / e^{-8\lambda t} = e^{7\lambda t} = e^1$.

$7\lambda t = 1 \Rightarrow t = 1/(7\lambda)$. Answer: $t = 1/(7\lambda)$.

🎯 PATTERN 9: Age of Rock / Carbon Dating

PYQ (multiple): Radioactive X with half-life $1.4 \times 10^9$ years decays to stable Y. Rock sample has X and Y in ratio 1:7. Age of rock?

Ratio X:Y = 1:7. So if $N$ = X now, originally $N_0 = N + 7N = 8N$ (Y came from X).

$N/N_0 = 1/8 = (1/2)^3$. So 3 half-lives have passed.

Age = $3 \times T_{1/2} = 3 \times 1.4\times10^9 = 4.2 \times 10^9$ years.

Nuclear Reactions — Fission, Fusion & Q-value Medium Frequency

Nuclear Fission

Heavy nucleus splits into medium-mass fragments + neutrons + energy. $^{235}$U + $^1_0$n → fission products + 2–3 neutrons + ~200 MeV. Slow (thermal) neutrons more effective. Chain reaction possible.

Nuclear Fusion

Light nuclei combine into heavier one + energy. $^2H + ^3H \to ^4He + n + 17.6$ MeV. Requires extremely high temperature (~$10^7$ K) to overcome Coulomb repulsion. Sun's energy source.

Q-value

$Q = (m_{reactants} - m_{products})c^2$ in MeV. $Q > 0$: exothermic (energy released). $Q < 0$: endothermic (energy needed). $Q = \Delta m \times 931.5$ MeV.

Critical Mass

Minimum mass of fissile material for self-sustaining chain reaction. Below critical mass: more neutrons escape than cause fission. At critical mass: exactly 1 neutron per fission causes next fission.

PYQ (mass-energy): Certain mass of hydrogen is changed to helium by fusion. Mass defect = 0.02866 u. Energy liberated per u is (1 u = 931 MeV):

Total energy = $0.02866 \times 931 = 26.7$ MeV.

Total mass involved in 4H → He: 4 u approximately. Energy per u = $26.7/4 \approx 6.675$ MeV/u.

NEET option check: answer should be ~6.67 MeV. This tests mass-energy calculation.

PYQ (reactor power): Power obtained in reactor using U-235 disintegration is 1000 kW. Mass decay of U-235 per hour?

$P = (\Delta m/t) \cdot c^2 \Rightarrow \Delta m/t = P/c^2$.

$P = 10^6$ W, $c = 3\times10^8$ m/s.

$\Delta m/t = 10^6 / (9\times10^{16}) = 1.11\times10^{-11}$ kg/s.

Per hour: $1.11\times10^{-11} \times 3600 = 4\times10^{-8}$ kg = $4\times10^{-5}$ g.

Chapter 3 — Last 5-Minute Recall

  • $R = R_0 A^{1/3}$, $R_0 = 1.2$ fm. Nuclear density = constant.
  • $\Delta m$ (u) × 931.5 = BE (MeV). For He-4: BE ≈ 28.3 MeV.
  • $N = N_0(1/2)^{t/T_{1/2}}$; Activity $A = \lambda N$; Mean life = $1.44 T_{1/2}$
  • α: A−4, Z−2. β⁻: A same, Z+1. β⁺: A same, Z−1. γ: no change.
  • Ionisation: α>β>γ. Penetration: γ>β>α.
  • BE/A max at Fe-56 (~8.8 MeV). Fusion (light) + Fission (heavy) both release energy.
  • Age problem: count half-lives from ratio $N_0/N = 2^n$.

Energy Band Theory — Classification of Solids Foundation

In isolated atoms, electrons have discrete energy levels. In a crystal with $N$ atoms, each level splits into $N$ closely spaced levels forming an energy band. The two most important bands are the valence band (highest filled) and the conduction band (lowest empty).

MaterialBand Gap ($E_g$)ConductivityExample
ConductorZero (bands overlap)Very high (~$10^7$ S/m)Cu, Ag, Al
Semiconductor~0.1–3 eV (small)IntermediateSi (1.1 eV), Ge (0.7 eV)
Insulator>3 eV (large)Very low (~$10^{-11}$ S/m)Diamond (5.5 eV), Glass
Intrinsic Semiconductor

Pure Si or Ge. At 0K: insulator. At room temp: thermal energy excites some electrons to conduction band. Equal holes and electrons: $n_e = n_h = n_i$.

n-type (Donor doping)

Add pentavalent impurity (P, As, Sb). 4 bonds with Si, 1 extra electron is free carrier. $n_e \gg n_h$. Electrons are majority carriers. Donor level is just below conduction band.

p-type (Acceptor doping)

Add trivalent impurity (B, Al, In). 3 bonds with Si, 1 hole created. $n_h \gg n_e$. Holes are majority carriers. Acceptor level is just above valence band.

⚠ Common Misconception — Doped Semiconductors

n-type semiconductor has excess electrons BUT it is electrically neutral overall — the donor atoms contribute both the free electron AND become positively charged ions. Similarly p-type is neutral. A doped semiconductor is NOT charged.

p-n Junction Diode High Frequency

When p-type and n-type semiconductors are joined, electrons diffuse from n to p and holes from p to n. This creates a depletion region with a built-in electric field (barrier potential ~0.3V for Ge, ~0.7V for Si) opposing further diffusion.

Forward Bias

p-side connected to +ve terminal. Reduces barrier potential. Depletion region narrows. Current flows above ~0.3V (Ge) or ~0.7V (Si). Low resistance.

Reverse Bias

p-side to −ve terminal. Increases barrier potential. Depletion region widens. Only tiny leakage current (minority carriers). High resistance. Acts as open switch.

Breakdown

At high reverse voltage: avalanche or Zener breakdown. Diode conducts heavily. Normal diodes damaged. Zener diodes designed to operate safely in breakdown.

Diode as Rectifier

Half-wave Rectifier

1 diode. Conducts only in positive half-cycle. Output: pulsating DC with 50% efficiency. Output frequency = input frequency.

Full-wave Rectifier (Bridge)

4 diodes in bridge. Conducts in both half-cycles. Better efficiency (~81%). Output ripple frequency = 2× input frequency.

Transistor — Amplifier & Switch Action High Frequency

A transistor (BJT) has three regions: Emitter (highly doped), Base (very thin, lightly doped), Collector (moderately doped). Two types: n-p-n and p-n-p. Emitter-base is forward biased, collector-base is reverse biased in active mode.

Transistor Current Relations
$I_E = I_B + I_C$ (KCL)
Current gain: $\beta = I_C/I_B$ (common emitter, CE)
$\alpha = I_C/I_E$ (common base, CB)
Relation: $\beta = \alpha/(1-\alpha)$; $\alpha = \beta/(\beta+1)$
Voltage gain (CE amp): $A_V = \beta \times R_L/R_i$
Transistor as Switch

When base current $I_B = 0$: transistor OFF (open switch, $I_C \approx 0$). When $I_B$ is sufficient: transistor ON (saturated, acts as closed switch). Digital circuits use this.

Transistor as Amplifier

Small change in base current causes large change in collector current. CE configuration gives both voltage and current amplification. $A_V \approx 10–1000$.

PYQ (pattern): For a transistor in CE configuration: $\beta = 50$, $I_B = 0.02$ mA, $R_C = 2$ kΩ. Find $I_C$, $I_E$, and voltage drop across $R_C$.

$I_C = \beta \times I_B = 50 \times 0.02 = 1$ mA.

$I_E = I_C + I_B = 1 + 0.02 = 1.02$ mA.

$V_{RC} = I_C \times R_C = 1\times10^{-3} \times 2\times10^3 = 2$ V.

🎯 PATTERN 11: Identify Gate from Circuit Diagram

NEET shows two diode/transistor circuits and asks which gate is implemented. Strategy: trace current path for all 4 input combinations (00, 01, 10, 11) and match with truth table.

Diode AND gate: Two diodes connected such that output is HIGH only when both inputs are HIGH. If either input is LOW, the diode conducts and pulls output LOW.

Diode OR gate: Two diodes connected such that output is HIGH when at least one input is HIGH. If both are LOW, output is LOW.

Transistor NOT gate: Single transistor in CE mode. When input is HIGH (transistor ON), output at collector is LOW. When input is LOW (transistor OFF), output is HIGH. Perfect inverter.

PYQ (pattern — appears every 2 years): In the circuit, two inputs A and B are given. For A=1, B=0 the output Y = 1. For A=1, B=1, Y = 0. Identify the gate.

Check against truth tables: Y=0 when A=1,B=1; Y=1 when A=1,B=0.

For NAND: (1,1)→0 ✓; (1,0)→1 ✓. This matches NAND behaviour. Answer: NAND gate.

Alternatively, check XOR: (1,1)→0 ✓; (1,0)→1 ✓. Also matches! Need more input combos to distinguish. For (0,0): NAND gives 1, XOR gives 0 — that would be the discriminating input.

Boolean Algebra — Key Identities NEET Uses

De Morgan's Laws
  • $\overline{A+B} = \bar{A}\cdot\bar{B}$ (NOR = AND of NOT)
  • $\overline{A\cdot B} = \bar{A}+\bar{B}$ (NAND = OR of NOT)
  • Used to build any gate from NAND/NOR
Basic Boolean
  • $A \cdot 0 = 0$; $A \cdot 1 = A$
  • $A + 0 = A$; $A + 1 = 1$
  • $A \cdot A = A$; $A + A = A$
  • $A \cdot \bar{A} = 0$; $A + \bar{A} = 1$
NAND Realisations
  • NOT: single NAND with A=B
  • AND: NAND + NOT
  • OR: NAND(NAND(A,A), NAND(B,B))
  • NOR: NAND(NAND(A,B)|NAND)
T
❌ Trap

"NAND output is 1 when both inputs are 1"

✓ Correct

NAND output is 0 ONLY when both inputs are 1. For all other combinations (00,01,10), output is 1. Think: NAND = NOT AND.

  • LED = forward. Photodiode = reverse. Solar cell = no bias. Zener = reverse.
  • $I_E = I_B + I_C$; $\beta = I_C/I_B$; $\alpha = I_C/I_E$; $\beta = \alpha/(1-\alpha)$.
  • AND: 1 only if both 1. OR: 0 only if both 0. NAND: 0 only if both 1. NOR: 1 only if both 0. XOR: 1 if different.
  • Universal gates: NAND and NOR.
