Uses of Dimensional Analysis

Principle of Homogeneity

Every term in a dimensionally correct equation must have the same dimensions. If $X = A + B - C$, then $[X] = [A] = [B] = [C]$.

Deriving formula: $T = k \cdot l^a g^b$ → match dimensions: $[T] = [L]^a[LT^{-2}]^b$ → $a+b=0$, $-2b=1$ → $b=-\frac{1}{2}$, $a=\frac{1}{2}$
So $T \propto \sqrt{l/g}$
⚠ Limitations of Dimensional Analysis

Cannot determine dimensionless constants (like $2\pi$ in $T = 2\pi\sqrt{l/g}$). Cannot distinguish between formulae with same dimensions. Cannot apply to functions like $\sin\theta$, $e^x$, $\log x$.

Error Analysis High Yield

Types of Errors

Absolute error: $\Delta a = |a_i - \bar{a}|$

Mean absolute error: $\overline{\Delta a} = \frac{\sum|\Delta a_i|}{n}$

Relative/Fractional error: $\frac{\overline{\Delta a}}{\bar{a}}$

Percentage error: $\frac{\overline{\Delta a}}{\bar{a}} \times 100\%$

Error Propagation Rules

Sum/Difference: $\Delta Z = \Delta A + \Delta B$

Product/Division: $\frac{\Delta Z}{Z} = \frac{\Delta A}{A} + \frac{\Delta B}{B}$

Power: if $Z = A^p B^q / C^r$, then

$\frac{\Delta Z}{Z} = p\frac{\Delta A}{A} + q\frac{\Delta B}{B} + r\frac{\Delta C}{C}$

NEET trap: For $Z = A^2 B / \sqrt{C}$, the max relative error = $2\frac{\Delta A}{A} + \frac{\Delta B}{B} + \frac{1}{2}\frac{\Delta C}{C}$. The power acts as a multiplier on the relative error.

Significant Figures

Rules for Significant Figures

All non-zero digits are significant. Zeros between non-zero digits are significant. Leading zeros are NOT significant (0.0023 → 2 sig figs). Trailing zeros after decimal ARE significant (2.300 → 4 sig figs). Trailing zeros without decimal are ambiguous.

In addition/subtraction: result has digits up to smallest decimal place of operands.

In multiplication/division: result has sig figs equal to the least in any operand.

Chapter 2 · Kinematics I
Motion in a Straight Line

Basic Kinematic Quantities

★★★ Core Chapter — 3–4 Qs NEET
Scalar vs Vector

Distance (scalar): total path length, always $\geq 0$

Displacement (vector): shortest distance between initial and final position, can be negative

Speed (scalar) = distance/time

Velocity (vector) = displacement/time

Average vs Instantaneous

$v_{avg} = \dfrac{\Delta x}{\Delta t} = \dfrac{x_2 - x_1}{t_2 - t_1}$

$v_{inst} = \lim_{\Delta t \to 0}\dfrac{\Delta x}{\Delta t} = \dfrac{dx}{dt}$

$a_{avg} = \dfrac{\Delta v}{\Delta t}$

$a_{inst} = \dfrac{dv}{dt} = v\dfrac{dv}{dx}$

Average Speed vs Average Velocity

Average speed = total distance / total time. Average velocity = total displacement / total time. They are equal only for unidirectional motion. For a round trip, average velocity = 0 but average speed ≠ 0.

For two equal halves of distance at speeds $v_1$ and $v_2$: $v_{avg} = \dfrac{2v_1v_2}{v_1+v_2}$ (harmonic mean)

For two equal halves of time at speeds $v_1$ and $v_2$: $v_{avg} = \dfrac{v_1+v_2}{2}$ (arithmetic mean)

Equations of Motion (Uniform Acceleration) Must Know

The Big 3:
$v = u + at$
$s = ut + \dfrac{1}{2}at^2$
$v^2 = u^2 + 2as$

Derived 4th: $s = \dfrac{u+v}{2} \cdot t$ (displacement = average velocity × time)
Displacement in nth second: $s_n = u + \dfrac{a}{2}(2n-1)$

$s_n$ formula: Displacement in the $n$-th second (not in $n$ seconds!) $= u + \frac{a}{2}(2n-1)$. Used frequently in NEET for "distance in 3rd second" type questions.

When to use which equation: Use $v = u + at$ when $s$ is not needed. Use $v^2 = u^2 + 2as$ when $t$ is not needed. Use $s = ut + \frac{1}{2}at^2$ when $v_{final}$ is not needed.

Ratio of distances in 1st, 2nd, 3rd…n seconds (starting from rest): $1 : 3 : 5 : 7 : \ldots : (2n-1)$ — odd number ratio

Ratio of total distances after 1st, 2nd, 3rd…n seconds: $1 : 4 : 9 : 16 \ldots$ — perfect squares

Motion Graphs Highest Freq

x–t Graph (Position-Time)

Slope = instantaneous velocity

Straight line → uniform velocity ($a=0$)

Curved (parabola upward) → uniformly accelerated

Horizontal line → body at rest

Slope negative → moving in −ve direction

v–t Graph (Velocity-Time)

Slope = acceleration

Area under curve = displacement

Straight line with +ve slope → uniform acceleration

Horizontal line → uniform velocity ($a=0$)

Area below x-axis = negative displacement

a–t Graph (Acceleration-Time)

Area under curve = change in velocity $\Delta v$

Horizontal line → constant (uniform) acceleration

⚠ NEET Traps in Graphs

Steep slope on x-t graph ≠ high acceleration — it means high velocity. A curved x-t graph means changing velocity (acceleration ≠ 0). Two objects meeting on x-t graph = same position, NOT same velocity. Highest point on v-t graph ≠ maximum displacement — displacement = area.

Free Fall & Vertical Motion

For free fall (taking downward as +ve, $u=0$):
$v = gt$
$h = \dfrac{1}{2}gt^2$
$v^2 = 2gh$
Time to fall height $h$: $t = \sqrt{\dfrac{2h}{g}}$
$g = 9.8 \text{ m/s}^2 \approx 10 \text{ m/s}^2$ (for NEET calculations)
Ball Thrown Upward

Time to reach max height: $t_{up} = \dfrac{u}{g}$

Maximum height: $H = \dfrac{u^2}{2g}$

Time to return: $t_{total} = \dfrac{2u}{g}$

Speed on return = speed of projection $u$

At max height: $v = 0$, acceleration $= g$ (downward)

Ball Thrown Down / Dropped

$v = u + gt$ (u = initial downward velocity)

$h = ut + \dfrac{1}{2}gt^2$

Dropped: $u = 0$, $h = \dfrac{1}{2}gt^2$

Ratio of distances: 1:3:5 (successive seconds)

Relative Motion

Relative Velocity Formula

$\vec{v}_{AB} = \vec{v}_A - \vec{v}_B$ (velocity of A relative to B)

Same direction: $v_{rel} = v_A - v_B$

Opposite directions: $v_{rel} = v_A + v_B$

Time to meet / cross: $t = \dfrac{d}{v_{rel}}$

Train problems: Time to cross a stationary object = $\frac{L_{train}}{v_{train}}$. Time to cross another moving object = $\frac{L_1 + L_2}{v_{rel}}$.

🔴 High Odd-number ratio for distances from rest
Body starting from rest (u=0): distances in 1st, 2nd, 3rd... successive seconds are in ratio 1 : 3 : 5 : 7 (odd numbers). Total distances after 1, 2, 3... seconds → 1 : 4 : 9 (perfect squares).
✓ Use: "Ball dropped, find ratio of distances in 3rd and 1st second" → 5:1 ✗ Fails: Only valid when u = 0 (starts from rest)

✓ Use: All v-t area problems ✗ Fails for distance: must take |area| for each segment separately

✓ Use: When both initial and final velocity are known ✗ Fails: Variable acceleration — then $v_{avg} = \Delta s/\Delta t$ is the only valid formula

🧠 Layer 5 — Derived Formula Bank

🔴 High — Combined kinematics forms for standard setups
Ball thrown up, returns to same level: total time $T = 2u/g$; max height $H = u^2/2g$; speed on return $= u$ (unchanged).
Dropped from height $h$: velocity at ground $v = \sqrt{2gh}$; time $t = \sqrt{2h/g}$.
Two bodies meet: set $s_1(t) = s_2(t)$, solve for $t$. Always use same reference point for both.

Time for overtaking: $t = \frac{d}{v_{rel}} = \frac{d}{v_A - v_B}$ (same direction)

$e = \dfrac{\text{relative speed of separation}}{\text{relative speed of approach}}$

Wave Fundamentals Core

★★★ Waves — 4–5 Qs/year NEET
Wave equation: $y(x,t) = A\sin(\omega t - kx + \phi)$
$k = \dfrac{2\pi}{\lambda}$ (wave number), $\omega = 2\pi f$
Wave speed: $v = \dfrac{\omega}{k} = f\lambda = \dfrac{\lambda}{T}$
Velocity of particle: $v_{particle} = \dfrac{\partial y}{\partial t}$
Acceleration of particle: $a_{particle} = \dfrac{\partial^2 y}{\partial t^2}$
Types of Waves

Transverse: displacement ⊥ propagation (light, string)

Longitudinal: displacement ∥ propagation (sound in air)

Sound: longitudinal in all media

Light: transverse (electromagnetic)

Seismic P-waves: longitudinal; S-waves: transverse

Speed on a String

$v = \sqrt{\dfrac{T}{\mu}}$ ($T$ = tension, $\mu$ = linear mass density)

$\mu = m/L$ (mass per unit length)

$v \propto \sqrt{T}$ and $v \propto 1/\sqrt{\mu}$

Thinner/lighter string at same tension → faster waves

Intensity & Power

Intensity: $I = 2\pi^2 f^2 A^2 \rho v$

$I \propto A^2$ (intensity ∝ amplitude squared)

$I \propto f^2$ (intensity ∝ frequency squared)

For a point source: $I \propto 1/r^2$

Intensity level: $\beta = 10\log_{10}(I/I_0)$ dB, $I_0 = 10^{-12}$ W/m²

Speed of Sound

Newton's formula (isothermal): $v = \sqrt{B/\rho}$ ($B$ = bulk modulus)
Laplace's correction (adiabatic): $v = \sqrt{\gamma P/\rho} = \sqrt{\gamma RT/M}$
At STP in air: $v \approx 332$ m/s; at 0°C: $v = 332$ m/s
$v_T = v_0\sqrt{T/273} \approx 332 + 0.6t$ m/s (where $t$ in °C)
Sound in Different Media

$v_{solid} > v_{liquid} > v_{gas}$ (in general)

$v \propto \sqrt{T}$ (in ideal gas, not pressure)

$v$ independent of pressure at constant temp

$v \propto 1/\sqrt{M}$ (lighter gas → faster)

Water: ~1480 m/s; Steel: ~5960 m/s

Pitch ∝ frequency (high-pitched = high-frequency)

Loudness ∝ intensity ∝ $A^2$ (decibels)

Quality (timbre) = waveform shape (overtones)

Doppler Effect NEET Favourite

General Doppler formula:
$f_{obs} = f_0 \cdot \dfrac{v + v_o}{v - v_s}$ (observer moving toward source, source moving toward observer)

Sign convention: $v_o$ positive if observer moves toward source; $v_s$ positive if source moves toward observer

Observer moving, source stationary: $f_{obs} = f_0\dfrac{v \pm v_o}{v}$
Source moving, observer stationary: $f_{obs} = f_0\dfrac{v}{v \mp v_s}$
Cases Summary

Source approaches: $f_{obs} = f_0 \dfrac{v}{v-v_s}$ ($> f_0$)

Source recedes: $f_{obs} = f_0 \dfrac{v}{v+v_s}$ ($< f_0$)

Observer approaches: $f_{obs} = f_0 \dfrac{v+v_o}{v}$ ($> f_0$)

Observer recedes: $f_{obs} = f_0 \dfrac{v-v_o}{v}$ ($< f_0$)

Applications

Approaching → pitch rises; Receding → pitch falls

Police radar: uses reflected Doppler shift

Red shift (Hubble): galaxies moving away → longer $\lambda$

Sonic boom: when $v_s = v$ (source at Mach 1)

Mach number: $M = v_{body}/v_{sound}$

⚠ Doppler Traps

Doppler effect is for frequency, not wavelength, when observer moves. Wavelength ahead of moving source: $\lambda' = (v-v_s)/f$. Behind: $\lambda' = (v+v_s)/f$. Observer moving does NOT change wavelength, only apparent frequency.

Beats & Superposition

Beats

Beat frequency: $f_{beat} = |f_1 - f_2|$

Beat period: $T_{beat} = 1/f_{beat}$

Max intensity at: $f = (f_1+f_2)/2$ (average frequency)

Beats heard if $f_{beat} \leq 10$ Hz (audible range of beats)

Used to tune musical instruments

Superposition Principle

$y_{total} = y_1 + y_2$ (algebraic sum)

Constructive interference: $\Delta\phi = 0, 2\pi, 4\pi,\ldots$ (even multiples of $\pi$)

Destructive interference: $\Delta\phi = \pi, 3\pi, 5\pi,\ldots$ (odd multiples of $\pi$)

Path diff for constructive: $\Delta x = n\lambda$

Path diff for destructive: $\Delta x = (2n-1)\lambda/2$

Quick Beats Shortcut
If two strings have beat frequency 4 Hz and on tightening one, beat frequency reduces to 2 Hz → the tightened string had the lower frequency. (Tightening raises frequency; if beat decreases, you were below the other.)
✓ Use: Tuning problems
